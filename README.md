# serial
easy to use wrapper around screen to connect to a tty

    $ serial
    Select Device to connect:
    [0] tty.Bluetooth-Incoming-Port
    [1] tty.Bluetooth-Modem
    (default = 0) >
    [0] 9600 - often arduino
    [1] 115200 - cubieboard, buspirate
    (default = 1) >
